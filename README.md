# 体重小本本 (Weight Notebook)

一款清新手绘风格的体重追踪网页应用，帮助记录和分析体重变化。

## 功能特点

- ✨ **清新手绘UI** - 可爱的手绘风格界面
- 📊 **数据可视化** - 体重曲线图、BMI图表、日历热力图
- 🎯 **目标管理** - 设置目标体重、计算时间线
- 🏆 **成就系统** - XP等级、徽章、里程碑
- 📸 **进度照片** - 上传前后对比照片
- 📈 **趋势分析** - 体重稳定性分析
- 💾 **数据导出** - 支持CSV和JSON格式
- 🎨 **8种主题** - 多种配色方案可选
- 📱 **响应式** - 支持手机、平板、电脑

## 在线使用

访问：https://rickyrelive.github.io/weight_loss/

所有数据都存储在本地浏览器中，隐私安全。

## 本地使用

直接打开 `index.html` 文件即可在浏览器中使用。

## GitHub Pages 部署

### 首次部署步骤：

1. **在GitHub仓库设置中启用Pages**
   - 访问仓库的 Settings → Pages
   - Source 选择：GitHub Actions
   - 点击 Save

2. **推送代码**
   ```bash
   git push origin main
   ```

3. **等待部署完成**
   - 访问 Actions 标签页查看部署进度
   - 部署成功后会显示绿色勾号

### 后续更新

只需推送代码到 main 分支即可自动部署：
```bash
git add .
git commit -m "更新描述"
git push origin main
```
